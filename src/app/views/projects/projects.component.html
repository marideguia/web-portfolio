Deep Learning: Vehicle Occupant Classification

Our team of five built and trained a deep learning model using Python and the Keras libraries to classify images of vehicle occupants into seven categories using interpretable and explainable deep learning methods. We were able to improve the initial accuracy of the model by an average of 11.7% using added datasets and data augmentation.
