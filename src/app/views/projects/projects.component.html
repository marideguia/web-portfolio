Vehicle Occupant Classification: Deep Learning

A deep-learning model built using Python and the Keras library to classify images of vehicle occupants. The model's average initial accuracy was raised from 65.6% to 77.3% (an improvement of 11.7%) using added datasets and data augmentation. Grad-CAM was then implemented to illustrate the model's interpretable and explainable results.
